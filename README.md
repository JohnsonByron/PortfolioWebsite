# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my projects, skills, and contact information.

## 🚀 Live Demo
[View Website](https://JohnsonByron.github.io/PortfolioWebsite)

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ✨ Features
- Responsive design (mobile-friendly)
- Smooth scrolling navigation
- Projects section
- About Me section
- Contact form
- Clean UI design

## 📁 Project Structure
PortfolioWebsite/
│── index.html
│── style.css
│── script.js
│── assets/
└── README.md
## ⚙️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/JohnsonByron/PortfolioWebsite.git

2. Open index.html in your browser

No additional setup required.
